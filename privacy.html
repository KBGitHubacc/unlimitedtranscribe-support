Privacy Policy

Last updated: May 21, 2026

UnlimitedTranscribe is designed to transcribe audio using either local on-device Whisper models or a user-provided OpenAI API key.

Audio Processing

When using local Whisper transcription, audio is processed on the user’s device and is not sent to an external transcription service.

When using OpenAI transcription, audio is sent to OpenAI for processing using the API key provided by the user.

API Keys

If the user chooses to use OpenAI processing, the OpenAI API key is stored securely on the device using Apple Keychain.

Local Data

UnlimitedTranscribe may store temporary audio artifacts, intermediate transcripts, final transcripts, settings, and job information locally on the device.

Users can delete local app data from within the app.

Data Collection

UnlimitedTranscribe does not sell user data.

UnlimitedTranscribe does not use user audio or transcripts for advertising.

UnlimitedTranscribe does not require an external account or login.

Third-Party Services

If OpenAI processing is enabled by the user, audio and related request data are processed by OpenAI according to OpenAI’s applicable terms and privacy policy.

Contact

For privacy questions, contact: support@krisz.co.uk
